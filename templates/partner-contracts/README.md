# 파트너 계약서 템플릿

이 폴더의 DOCX는 **등급별 원본 계약서**를 둡니다.

- `silver.docx`
- `gold.docx`
- `platinum.docx`

애플리케이션은 원본을 복제한 뒤 치환 토큰만 교체합니다.
HTML→DOCX 재작성, 조항 문구 변경, Storage 저장은 하지 않습니다.

## 치환 토큰 (템플릿에 포함)

- `OOOO` / `OOOOO` / `OOOOOO` → 계약서 표기 회사명
- `【상호】` `【대표이사】` `【사업자등록번호】`
- `【계약일】` `【계약시작일】` `【계약종료일】`

실제 표준 계약서 원본이 있으면 위 파일들을 덮어쓰세요.
플레이스홀더가 원본 서식의 해당 위치에 들어가 있어야 합니다.
